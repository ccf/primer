This guide gets you from zero to a running Primer instance with live session data in about five minutes.

## Prerequisites

- Python 3.12+
- At least one AI coding tool: [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [Codex CLI](https://github.com/openai/codex), or [Gemini CLI](https://github.com/google-gemini/gemini-cli)

## Step 1: Install

```bash
pip install primer
```

This installs the `primer` CLI, FastAPI server, hook system, and MCP sidecar.

## Step 2: Initialize

```bash
primer init
```

Creates a local SQLite database and config at `~/.primer/`. For production, you can [configure PostgreSQL](/docs/configuration/) later.

## Step 3: Start the server

```bash
primer server start
```

The server starts in the background on `http://localhost:8000`. Open it in your browser — you'll see the dashboard (empty for now).

## Step 4: Install hooks

For **Claude Code** (recommended starting point):

```bash
primer hook install
```

For **Gemini CLI**:

```bash
primer hook install --agent gemini
```

For **Codex CLI** (no hook system, uses sync):

```bash
primer sync --watch
```

Check what's installed:

```bash
primer hook list
```

## Step 5: Register yourself

```bash
primer setup
```

Creates your engineer profile. Name and email are pulled from `git config` by default.

## Step 6: Use your AI tool

Start a Claude Code session, do some work, and end it. Within seconds, the session appears in your dashboard with cost estimates, tool usage, and outcome classification.

After a few sessions, you'll start seeing patterns: activity trends, friction signals, and outcome distribution on the Organization page.

## Step 7 (optional): MCP sidecar

Add the Primer MCP server so Claude can query your stats mid-session:

```bash
primer mcp install
```

Now you can ask Claude things like "What's my success rate this week?" or "What friction am I hitting most?" and get live answers from your own data.

## Verify everything

```bash
primer doctor
```

Checks your database, hook installation, server status, and MCP connection.

## Step 8 (optional): Seed demo data

If you want to explore the full dashboard before collecting real data:

```bash
python scripts/seed_data.py
```

This populates the database with a realistic 25-person engineering organization across 6 teams.
